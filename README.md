# Spring Boot Pagination Project

A simple Spring Boot simple project demonstrating **pagination** and **sorting** using Spring Data JPA.

---

## Features

- Retrieve paginated list of employees
- Sort by any column (id, name, joiningDate, etc.)
- RESTful API endpoints
- Simple Spring Boot setup with mysql

---

## Technologies Used

- Java 17
- Spring Boot 3.x
- Spring Data JPA
- mysql
- Maven
- Git/GitHub
